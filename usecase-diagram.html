Room Management

Create Room - Generate a new collaboration room with unique ID
Join Room - Enter existing room using Room ID and username
Leave Room - Disconnect from current collaboration session
Generate Room ID - Create unique UUID for room sharing
Copy Room ID - Copy room identifier to clipboard

File Management

Create File - Add new file to project structure
Edit File - Modify file content with real-time sync
Delete File - Remove file from project
Rename File - Change file name
Create Directory - Add new folder to organize files
Delete Directory - Remove directory and contents
Rename Directory - Change directory name
Open File - Open file in editor tab
Close File - Close file tab
Download Project - Export entire project as ZIP

Real-time Collaboration

View Cursor Positions - See where other users are editing
View Typing Indicators - Know when others are typing
See User Presence - View online/offline status
Synchronize Code Changes - Real-time code updates across all users

Code Execution

Execute Code - Run code in multiple programming languages
View Execution Output - See program output and results
Handle Execution Errors - Display compilation/runtime errors

Communication

Send Chat Message - Send message to room participants
Receive Chat Messages - Get messages from other users
View Chat History - See conversation within session

Collaborative Drawing

Draw on Canvas - Create drawings and sketches
View Others' Drawings - See collaborative drawings in real-time
Switch Drawing Mode - Toggle between code and drawing views

AI Services

Generate Code - AI generates code based on prompts
Analyze Code for Errors - AI detects bugs and issues
Get Code Suggestions - AI provides improvement recommendations
Ask Coding Questions - Get answers about code and programming

Customization

Change Theme - Switch between color schemes
Adjust Font Size - Modify editor font size
Change Font Family - Select preferred coding font
Configure Settings - Manage all editor preferences
